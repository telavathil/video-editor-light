# Vacation Video Editor — Project Rules

## Design is the Source of Truth

The Pencil.dev design file (`design/video-editor-light.pen`) is the authoritative reference for all UI decisions.

- **The app must follow the design, not the other way around.**
- When the app and design differ, the design is correct and the app needs to be updated.
- Do NOT modify the design to match the current state of the app unless explicitly instructed.
- Design gaps (missing features not yet in the design) should be added to the design first, then implemented in the app.
